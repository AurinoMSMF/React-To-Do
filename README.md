# To-do

## Recursos:

- É possivel criar, visualizar detalhes, concluir e excluir uma tarefa
- Gráfico dinâmico de exibição do desempenho de conclusão das tarefas nos ultimos 7 dias

## Qualidade e Testes

Este projeto possui testes automatizados E2E com Cypress focados nos fluxos críticos:

- Criação de tarefa
- Visualização de detalhes da tarefa
- Conclusão de tarefa
- Exclusão de tarefa

Os testes são executados automaticamente via GitHub Actions a cada push.
